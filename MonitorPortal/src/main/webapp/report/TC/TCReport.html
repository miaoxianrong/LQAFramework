[Result of Pre Process] During Pre Process at TC, if ploc.pseudo.i18n is existed in TC FTP, timestamp is retrieved from the contents of ploc.pseudo.i18n and unsynched unit is checked. If not existed, blank is displayed.
 
[Pseudo Localize] You can see 1) whether Pseudo Localize in i18nStudio is executed or not, 2) and the same English resource files are at least used in both Pseudo Localize and Creating Turnover Package.
Pseudo Localized At : This shows when Pseudo Localize is done by i18nStudio. Timestamp is extracted from the contents (Last Updated At line) of ploc.pseudo.i18n file.
# of Unsynched Unit : This number should be 0. This number shows a gap between ploc.pseudo.i18n from Pseudo Localize and translatable i18n from Turnover Package. More than 0 means that English resource files are different, Templates are different, etc.
 
[Result of Post Process] During Post Process at TC, if UI Context Zip is existed in TC FTP, its contents are analyzed. If not existed, blank is displayed.
 
[Translation Unit Mapping] This data shows based on the number of Translation Unit in translatable i18n file.
Total : This shows the total Translation Unit Count in translatable i18n file.
New : This shows the Translation Unit Count which leverage status is New in translatable i18n file.
Fuzzy : This shows the Translation Unit Count which leverage Status is Fuzzy in translatable i18n file.
 
# of Unit is the number of Translation Unit in translatable i18n file.
# of Mapped Unit is the number of Translation Unit which are mapped to a screen.
Mapped % is the percentage of # of Mapped Unit / # of Unit. 100% means all Translation Units have Screen.
 
[UI Context Zip from LQA Framework] This data shows based on the contents of UI Context Zip file.
[Screenshot] This data shows screenshot files in UI Context Zip file.
# of Screen : This shows the number of screenshot files in UI Context Zip.
 
[Pseudo Unique ID] This data shows based on Pseudo Unique ID in text-to-screen.txt file in UI Context Zip file.
No of PUID : This shows the number of Pseudo Unique IDs in text-to-screen.txt.
No of PUID No Rep : This shows the number of Pseudo Unique IDs without Repetition in text-to-screen.txt.
 
[# of Element] This data shows based on each Element (= each Mapping) in text-to-screen.txt file in UI Context Zip file.
Total : This shows the number of total Elements in text-to-screen.txt.
With PUID : This shows the number of Element which has Pseudo Unique ID in text-to-screen.txt.
With PUID No Rep: This shows the number of Element which has Pseudo Unique ID without repetition in text-to-screen.txt.
Without Screen : 0 element and 0% are desirable. This shows the number of Element which does NOT have screenshot in text-to-screen.txt.
